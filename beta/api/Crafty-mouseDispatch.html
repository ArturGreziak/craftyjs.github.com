Crafty.mouseDispatch()

src/controls/inputs.js#L136-L151

Internal method which dispatches mouse events received by Crafty.

This method processes a native MouseEvent received by Crafty.stage.elem, augments it with additional properties and dispatches it to the closest (visible & Mouse-enhanced) entity to the source of the event (if available).

This method also updates Crafty.lastEvent.
